# TLS для `proxy-nginx`

Сюда кладите файлы (имена зашиты в конфиг Nginx):

| Файл | Назначение |
|------|------------|
| `catalogbooks.crt` | сертификат (желательно полная цепочь до корневого УЦ) |
| `catalogbooks.key` | приватный ключ |

Файлы монтируются в контейнер как `/etc/nginx/ssl/` (только чтение).

**Не коммитьте** `.key` / `.crt` в git (см. корневой `.gitignore`).

Сейчас **и порт 80, и 443** отдают одно и то же приложение; **редиректа HTTP → HTTPS нет**. Позже его можно добавить отдельным `server { listen 80; return 301 https://...; }` в `proxy-nginx/conf.d/default.conf`.
